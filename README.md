# shadcn/ui playroom

A small React design system playground built with [Playroom]() and (shadcn/ui)[https://ui.shadcn.com/].

Install...

```bash
npm install
```

Play locally...

```bash
npm run playroom:start
```

Build to deploy for remote...

```bash
npm run playroom:build
```
